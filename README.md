# Inverse Optimization for Niche Market Demand Estimation

This project developed a KKT-based inverse optimization framework to estimate consumer demand parameters in niche markets. Modeled seller revenue as a nonlinear forward problem, inverted KKT conditions to recover price sensitivity θ, and validated on a jewelry boutique dataset with <6% estimation error under sensitivity analysis.
